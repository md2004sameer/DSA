Algorithm Explanation

  1. Move n-1 disks from source to auxiliary (recursive call)
  2. Move the bottom disk from source to destination
  3. Move n-1 disks from auxiliary to destination (recursive call)

Algorithm Code with Line-by-Line Tracking

function moveDisks(n, source, destination, auxiliary) { if (n === 1) { // Base case: Move the single disk directly moveDisk(1, source, destination); return; } // Step 1: Move n-1 disks from source to auxiliary moveDisks(n-1, source, auxiliary, destination); // Step 2: Move the nth disk from source to destination moveDisk(n, source, destination); // Step 3: Move n-1 disks from auxiliary to destination moveDisks(n-1, auxiliary, destination, source); }
